Introduction to Lua Programming

What is Lua?

Lua is a lightweight, high-level programming language designed for embedded systems, scripting, and general-purpose programming. It was created in 1993 by Roberto Ierusalimschy, Luiz Henrique de Figueiredo, and Waldemar Celes at the Pontifical Catholic University of Rio de Janeiro, Brazil.

Key Features of Lua:

  • Lightweight and fast execution
  • Simple and expressive syntax
  • Powerful data structures, including tables
  • Dynamic typing
  • Automatic memory management (garbage collection)
  • First-class functions and closures
  • Extensibility through C API

Hello World in Lua:

print("Hello, World!")

Data Types and Variables:

Lua has eight primitive data types:

  • nil: represents the absence of value
  • boolean: represents true or false
  • number: represents numeric values
  • string: represents sequences of characters
  • function: represents executable code
  • userdata: represents arbitrary C data
  • thread: represents independent threads of execution
  • table: represents associative arrays

To declare a variable in Lua, use the local keyword:

local message = "Hello, Lua!"
